# Accessibility Copilot

Workspace de análisis de accesibilidad para webs públicas.

## Stack
- Frontend: Angular
- Backend: Java + Spring Boot
- Scanner: Node.js + Playwright + axe-core
- DB: PostgreSQL
- Infra local: Docker Compose

## MVP
- Crear proyecto
- Lanzar scan de una URL
- Detectar incidencias automáticas
- Guardar resultados
- Visualizar issues con severidad y detalle
- Generar sugerencias de fix para Angular
